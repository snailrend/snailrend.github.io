# hugo的安装

# 解压软件包

解压软件包，可以使用命令行，可以使用图形界面。 图形界面直接邮件解压就好了，这里说一下命令行。

使用一下命令可以将文件解压出来

gunzip hugo*.gz
tar xvf hugo*.tar

# 安装软件包

hugo 的安装比较简单与go 语言地编译器一样是解压就能用。 我习惯把这类软件放在/usr/local目录下

mv hugo* /usr/local

设置环境变量 这边我是使用修改profile文件的方法,优点是全用户有效

打开profile文件

vim /etc/profile

在最后一行添加

exprot PATH=$PATH:/usr/local/hugo

然后用下面的命令更新文件

source /etc/profile

至此hugo 就安装完成
